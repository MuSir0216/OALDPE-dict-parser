# OALDPE-dict-parser

## About OALDPE

This program parses the **OALDPE En-Cn V2024.8.10** dictionary.

The dictionary is obtained from:  https://forum.freemdict.com/t/topic/25795

The official website of the dictionary is:  https://www.oxfordlearnersdictionaries.com

To respect and protect the copyright of the dictionary, the dictionary files are **not distributed in this repository**.
